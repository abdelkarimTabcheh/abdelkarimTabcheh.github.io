Welcome to my blog! In this post, I’ll be sharing the key insights and experiences I gained from the CS50 course, along with an in-depth look at my final project.

What is CS50?#

Among all programming courses, CS50 stands out as one of the most popular, particularly for beginners. In just 10 weeks, you’ll be introduced to a wide array of topics, from Scratch to C, Python, SQL, and both frontend and backend development. The course challenges you with weekly problem sets and culminates in a final project that showcases what you’ve learned.

CS50 Weeks Briefly#

Week 0#

This week covers some fundamental computer science concepts such as binary, ASCII, and image representation. You’ll start with a gentle introduction to programming using Scratch, a visual language that helps ease you into the world of coding.

Week 1 - Week 5#

These weeks are dedicated to mastering C programming. You’ll begin with the basics, including data types, loops, pointers, and file handling, before moving on to more advanced topics like algorithms and data structures. Expect to tackle complex problems involving linked lists, trees, hash tables, and tries. The problem sets during these weeks are the most challenging of the course.

Week 6#

After diving deep into low-level programming with C, you’ll shift gears to learn Python, a high-level language known for its simplicity and ease of use. This week focuses on Python’s features, such as dictionaries and CSV file handling, which are more cumbersome in C but straightforward in Python.

Week 7#

Building on your understanding of CSV files from Week 6, this week introduces you to relational databases and SQL. You’ll explore the basics of SQLite and gain practical experience in efficiently storing and retrieving data, which will be both enlightening and enjoyable.

Week 8#

This week dives into the essentials of web development. You’ll learn about HTTP, TCP/IP, and how to create a basic website using HTML, CSS, and JavaScript, incorporating Bootstrap to enhance the design and functionality.

Week 9#

The final week of the course is dedicated to building a fully functional website using Python and Flask. You’ll apply the skills you’ve accumulated from previous weeks, including working with SQLite, HTML, and JavaScript, to create a comprehensive web application.

My Final Project: To-Do List App with Flask#

This is my final project for CS50’s Introduction to Computer Science, a task management website developed using Python and Flask. The application is designed to help users efficiently record and manage their tasks.

Project Overview#

The site features five main pages:

  1. Register Page

    • Description: Users can create a new account by filling out a registration form.
    • Fields:
      • Name: The user’s full name.
      • Username: The application checks for existing usernames and only accepts lowercase characters. Numbers and symbols are not allowed.
      • Password: The password is validated to ensure it matches the confirmation field.
    • Screenshot:

  2. Login Page

    • Description: Users can log in by entering their username and password. The application verifies credentials and creates a session for the user.
    • Screenshot:

  3. Main Screen

    • Description: Upon logging in, users are directed to the main screen, where they can view and manage tasks for the current date or any other date. Features include:

      • Adding New Tasks: Users can add new tasks for the current day or a different date. The “New Task” option opens a form with fields for task name and date.
        • Screenshot:
      • Prioritizing Tasks: Tasks can be moved to the top of the list for higher priority.
      • Deprioritizing Tasks: Completed tasks can be moved to the bottom of the list.
      • Marking Tasks as Done: Tasks can be marked as completed.
        • Screenshot:
      • Reopening Tasks: Tasks marked as done can be reopened and edited.
      • Editing Tasks: Modify task details such as name or date using the same form used for adding tasks.
      • Viewing Tasks: Browse tasks by date, show only undone tasks, or view all daily tasks.
      • Deleting Tasks: Remove tasks permanently from the database.

    • Screenshot:

  4. Logout

    • Description: Users can log out of the application by selecting the “log out” option in the page header.

Technologies Used#

  • SQL: Used for managing the database, including user accounts and tasks. Passwords are securely hashed in the database to enhance security.
  • Python: The primary language used for developing the application logic.
  • Flask: A lightweight web framework that handles the application’s routing and server-side functionality.

Security and User Management#

  • Authentication: Each route checks if the user is authenticated. Once logged in, users remain logged in until they log out.
  • User-Specific Data: Each user can only view their own tasks. The application uses foreign keys in the database to associate tasks with users.

Conclusion#

Thank you for exploring my To-Do List application. I hope this project demonstrates the practical application of the skills I acquired during CS50. If you have any questions or feedback, feel free to reach out!

Enjoy using the app!

What I Learned from CS50 and My Reflections#

I want to extend my heartfelt gratitude to Harvard University, David Malan, and the entire CS50 team for making this exceptional course available online and free of charge. CS50 has been an incredible journey, and it has significantly shaped my understanding of programming and computer science.

What I Learned#

Throughout the course, I gained a solid foundation in programming and various computer science concepts. The problem sets were particularly challenging, often mirroring real-life scenarios that demanded practical solutions. Tackling these problems was not always easy; there were moments of frustration, especially when facing persistent errors like “segmentation faults.” Despite these hurdles, each problem set pushed me to think critically and develop robust problem-solving skills.

Reflections#

After approximately 10 weeks of dedicated work, I successfully completed all the lectures and problem sets. The sense of accomplishment upon finishing the course was immense. I would also like to express my thanks to my teacher, whose guidance and support were instrumental in helping me navigate through the course.

I completed CS50 in the summer of 2020, just before starting my second year of computer science at the Lebanese University. The knowledge and skills I acquired have been invaluable as I continued my studies and professional growth.

Thank you once again to everyone involved in making CS50 a transformative learning experience.
